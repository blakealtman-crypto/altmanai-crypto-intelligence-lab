# Compliance Notes

**This document is informational and does not constitute legal advice.**

## What This Project Is

An educational and research technology project focused on explaining blockchain, digital asset, and market-structure concepts.

## What This Project Is Not

- Not a registered investment adviser or broker-dealer
- Not offering securities, investment contracts, or managed accounts
- Not providing personalized financial, legal, or tax advice
- Not executing trades, holding custody of user funds, or managing wallets
- Not guaranteeing any outcome, return, or performance

## Operating Principles

1. No feature in this repository recommends buying, selling, or holding any specific asset.
2. No content should be published implying guaranteed returns or "signals."
3. Any future feature involving real financial data, user accounts, or monetization requires legal review before implementation.
4. All public communications about this project should be reviewed against `DISCLAIMER.md` before publishing.

## Jurisdictional Note

Digital asset regulation varies significantly by country and region and continues to evolve. This project makes no jurisdiction-specific compliance claims. Anyone building on top of this project for a commercial purpose should seek qualified legal counsel for their specific jurisdiction and use case.
